# Визуализатор

Визуализатор, который мы поставляем, по сути представляет из себя отдельное приложение, открывающее окно GLFW и отрисовывающее кадры с помощью визуализатора projectM.

Когда вы откроете его, он также откроет диалоговое окно для изменения опций визуализатора.

![Визуализатор](../assets/visualizer.png)

Вы можете расширить визуализатор на полный экран двойным нажатием на окно.

## ФПС

Вы можете установить желаемое количество ФПС для визуализатора. По-умолчанию, оно установлено на значение частоты развёртки окна, в котором визуализатор открывается.

## Ширина и высота сетки

Ширина и высота сетки контолирует вид текстур, вы можете поиграться с этими опциями.

## Загружать случайный пресет для каждого нового трека

_Чтобы эта функция работала, вы должны установить папку со случайными пресетами и убедиться в том, что она действительно содержит пресеты._

Когда проигрывается новый трек, визуализатор загрузит случайный пресет.

## Загрузить пресет

Открывает диалоговое окно, позволяющее вам выбрать `.milk` пресет.
