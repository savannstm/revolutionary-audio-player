# Система плейлистов

Как мы уже узнали, RAP использует систему плейлистов.

## Вкладки

У каждого плейлиста есть собственная вкладка. Вы можете открыть её контекстное меню с опциями нажав правую кнопку мыши. Оттуда, вы можете переименовать вкладку, изменить её цвет, или удалить другие вкладки.

![Меню вкладки](../assets/tab-menu.png)

Вы также можете активировать переименование вкладки дважды кликнув по её тексту.

Вкладки можно переносить, и вы можете свободно менять их порядок.

## Дерево

У каждого плейлиста, очевидно, есть дерево, содержащее все открытые треки.

![Дерево](../assets/tree.png)

### Заголовок

Вы можете изменять столбцы заголовка переключая опции в контекстном меню заголовка, открываемом с помощью правой кнопки мыши.

![Меню заголовка](../assets/tree-header-menu.png)

Столбцы заголовка также могут быть переупорядочены и растянуты.

### Обзор

Обзор дерева содержит открытые треки.

![Обзор дерева](../assets/tree-view.png)

По-умолчанию, элементы каждого обзора отсортированы по пути к треку. Вы можете изменить сортировку, кликнув нужный столбец заголовка. После цикла сортировки, сортировка вернётся к сортировке по пути к файлу.

Обзор также позволяет вам переупорядочить треки, и их новый порядок будет сохранён. Вы также можете сортировать треки по их порядку.

![Изменённый порядок](../assets/tree-reordering.gif)

Выбрать несколько треков для переупорядовачения или ~~удаления~~ (В РАБОТЕ) можно удерживая `Shift`/`Ctrl` во время выбора.

Каждый плейлист может иметь свой задний фон, который вы можете выбрать из файла изображения, вы также можете изменить прозрачность всего дерева.

Среди опций есть относящиеся к самим элементам обзора, включающие удаление треков, или проверку метаданных и обложек отдельных треков.

![Меню обзора](../assets/tree-view-menu.png)

### Поиск

Вы можете искать треки в дереве, нажав горячие клавиши `Ctrl + F`.

Это открывает поле ввода в правой верхней части дерева. По-умолчанию, поиск происходит по названию, но существуют префиксы свойств, которые можно использовать для поиска чего-то конкретного.

Они включают:

-   `title` - Поиск по названию
-   `artist` - Поиск по артисту
-   `no` - Поиск по номеру трека
-   `album` - Поиск по альбому
-   `albumartist` - Поиск по артисту альбома
-   `genre` - Поиск по жанру
-   `year` - Поиск по году/дате
-   `duration` - Поиск по продолжительности
-   `composer` - Поиск по композитору
-   `bpm` - Поиск по BPM
-   `discnumber` - Поиск по номеру диска
-   `comment` - Поиск по комментарию
-   `publisher` - Поиск по издателю
-   `bitrate` - Поиск по битрейту
-   `samplerate` - Поиск по частоте дискретизации
-   `channels` - Поиск по количеству каналов
-   `format` - Поиск по формату кодека
-   `path` - Поиск по пути к треку
-   `order` - Поиск по порядку в плейлисте

Например, вы можете найти артиста Death Grips с помощью `artist:Death Grips`.
